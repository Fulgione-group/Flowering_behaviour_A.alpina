#	Welcome!
###	

In this directory:

1)	flowering_behavior_greenhouse_noVern.R: 		analyses and code to produce figure for the flowering behavior experiment without vernalization
2)	flowering_behavior_greenhouse_withVern.R: 		analyses and code to produce figure for the flowering behavior experiment with vernalization
3)	flowering_behavior_greenhouse_ftAndDuration.R:		analyses and code to produce figure for the comparison between flowering time (no vernalization) and flowering duration (after vernalization)
