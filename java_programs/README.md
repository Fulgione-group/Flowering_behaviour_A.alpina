# Java programs 
###

Java programs assume that you have the /java/ folder from GitHub in your home directory, and that the directory structure is the same as in /java/. 
If not so, you will need to change some paths (e.g. to the libraries in ./java/lib/)


To run these java programs, enter the ./java/projects directory and you will find:

###  pi_matrix_alpina.command
Java program to compute average pairwise differences across Arabis alpina seamples from whole genome data
